## ✨ Features

- 🖥️ **Ultimate OS Terminal**: A fully functional command-line interface mimicking a real Linux environment (`hd` shell).
- 🎮 **Snake Game Easter Egg**: Type `play snake` to reveal a hidden, playable retro Snake game directly inside the terminal output!
- 🌐 **3D Skills Visualizer**: Type `skills` to open a stunning, interactive 3D particle globe built with Three.js/React Three Fiber.
- 🎨 **Glitch & Glassmorphism Aesthetics**: A cyberpunk-inspired dark mode theme with glowing accents, realistic CRT scanlines, and buttery smooth animations using Framer Motion.
- 🛠️ **Real Commands**: Explore with commands like `help`, `whoami`, `projects`, `experience`, `education`, `pwd`, `date`, `echo`, `sudo`, `coffee`, `joke`, and a personalized `neofetch`.
- 📁 **File System Simulation**: Use `ls` and `cat` to read virtual markdown and log files containing my resume details.

## 🚀 Quick Start

To run this futuristic portfolio on your local machine, follow these steps:

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/harshdodiya58/Portfolio.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Portfolio
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```

5. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000) and watch the OS boot sequence!

## 💡 Tech Stack

This project was crafted with modern web technologies:

- **Framework**: Next.js 15 (App Router)
- **Library**: React 19
- **Styling**: Tailwind CSS + Custom CSS (for CRT/Glitch effects)
- **Animations**: Framer Motion
- **3D Graphics**: Three.js, React Three Fiber, React Three Drei
- **Deployment**: Vercel
